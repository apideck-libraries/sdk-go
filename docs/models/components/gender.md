# Gender

The gender represents the gender identity of a person.


## Values

| Name                 | Value                |
| -------------------- | -------------------- |
| `GenderMale`         | male                 |
| `GenderFemale`       | female               |
| `GenderUnisex`       | unisex               |
| `GenderOther`        | other                |
| `GenderNotSpecified` | not_specified        |
